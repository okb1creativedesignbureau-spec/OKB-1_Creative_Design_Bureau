# NACA 2R1 Bf 109 Wing — 3-Section Starter Dataset (Root + Mid + Tip)
Get three key NACA 2R1 airfoil sections from either Early (Emil) or Late (Gustav) Bf 109 variants. Includes geometry, plots, CSV, Excel, reference images, and DWG CAD files

## Overview
This starter bundle provides three (3) reconstructed NACA 2R1-derived airfoil sections from the historic Me 109 wing.

Choose between:

- Early Bf 109 Set (Emil)
- - 14.2% Root
- - 12.8% Mid
- - 11.0% Tip

- Late Bf 109 Set (Gustav)
- - 14.2% Root
- - 12.9% Mid
- - 11.35% Tip

This pack is ideal for users who need a partial dataset of the full wing geometry for aerodynamic studies, CAD modeling, or historical reconstruction work.

1) Geometry Data (All 3 Sections)

2) Excel Master File

- Thickness/chamber plots
- Overlay comparison chart
- Reconstruction notes

3) Reference Materials
- Diagrams and verification images

4) CAD Files
- DWG of each profile
